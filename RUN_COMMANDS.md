# 🎬 READY TO RUN - Copy & Paste Commands

## Step 1: Open Terminal

```
Windows: Win + R → cmd → Enter
Mac: Cmd + Space → terminal → Enter
Linux: Ctrl + Alt + T
```

## Step 2: Navigate to Project

```bash
cd d:\interview-buddy-ai-08-main
```

## Step 3: Run the Application

### First Time Only (Install Dependencies)
```bash
npm install
```

Wait for it to finish (2-3 minutes)...

### Every Time After (Start Dev Server)
```bash
npm run dev
```

```

  ➜  Local:   http://localhost:5173/
  ➜  press h to show help

```
http://localhost:5173/

And paste it in your browser, or just click it in the terminal.

## 🚨 If Terminal Shows Error

### Most Common Error: "npm not found"
**Fix:** Install Node.js from https://nodejs.org/

### Most Common Error: "Port 5173 in use"
**Fix:** Use different port:
```bash
npm run dev -- --port 3000
```

### Most Common Error: "Module not found"
**Fix:** Run this:
```bash
npm install
npm run dev
```
